## Description

[Nest v9.0.0](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Dev Note {NestJS CLI}

```bash
# generate module by terminal
$ nest g mo <name>

# generate controller by terminal
$ nest g co <name>

# generate service by terminal
$ nest g s <name>
```
